# Get started

ingos-server is based on .NET Core 3.1, so if you want to develop based on this repository, make sure to include all the following requirements item.



# Requirements

- IDE and SDK
  - Visual Studio 2019 or Visual Studio Code
  - .NET Core 3.1
- Database
  - MySQL 8.0
  - Redis
- ELK Stack
  - Elasticsearch
  - Logstash
  - Kibana
  - Elastic APM
- Docker
