# ingos-server

a product management software's back-end api project which adopt ddd thought and based on asp.net core web api.



# requirements

- IDE and SDK
  - Visual Studio 2019 or Visual Studio Code
  - .NET Core 3.1
- Database
  - MySQL 8.0
  - Redis
- Docker
